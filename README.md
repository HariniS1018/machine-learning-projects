# Machine Learning Projects

This repository contains multiple ML projects, each in its own folder:

1. **Amazon Alexa Reviews** – Classifying customer reviews on various products.
2. **Mobile Price Range Classification** – Predicting price categories for various mobiles.
3. **College Performance Analysis** – Data Analysis on Various colleges.
4. **Auto MPG EDA** - Exploratory Data Analysis and Model Decision.
5. **Stock Price Prediction** – Time series forecasting.
6. **Car Sales Prediction** - Understanding various Evaluation metrics for Linear Regression
7. **Credit Card Default Classification** - Understanding Naive Bayes Classifier on a highly imbalanced dataset.
8. **Diabetes Prediction** - Explore different Cross Validation Techniques to best fit model that predicts the presence of diabetes. 
9. **Model Benchmark** - Explore different classification models performance across different inbuilt datasets to understand the effects of the assumptions of the models on different scenarios.
10. **Handwritten Digits Classification** - Implementing SVM and Hyperparameter tuning using Grid search.
11. **Play Tennis Classification** - Understand Desicion Tree with ID3, CART, and bagging ensemble.
